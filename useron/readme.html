Installing Users Online
To install Users Online. Open online.php and edit the Variables. Then upload all the files inside this zip.
Then you need to CHMOD them. Here are the settings:
Filename Permisions
online.php
 755
online.html
 666


Users Online Template Codes and Other Info
Template Codes: (Case Sensitive)
Code Variable
{online}
 The total number of people on that page.
{page}
 The page that people are on.
{color}
 The color of that row. Requires Color Code from below to work.
{count}
 The number of people on online total.
{site}
 The loaded site's stats.
{link}
 The loaded site's link.
{record}
 The max amout of people at one time for that site.
{copyright}
 The copyright to my site. Leave this there and allways put it in your templates you do not have to use it as long as you put it in manually.


Date Codes: (Case Sensitive) [Function Reference]
Code Variable
m
 Month when the record was made.
j
 Day when the record was made.
Y
 Year when the record was made.
H
 Hour when the record was made.
i
 Minute when the record was made.
s
 Seconds when the record was made.


Color Code: (Case Sensitive)
[color "[[COLOR]]" "[[COLOR]]"]
Replace [[COLOR]] with your colors. To add more colors just add more:
< "[[COLOR]]">
Without the <>. Example:
[color "#FFFFFF" "#CCCCCC" "#999999"]

To Use:
<script src="http://www.domain.com/online.php?js=1"></script>
To view it if there is no link just do the same without &js=1 Example: http://www.domain.com/online.php
